High-Dimensional Data Analysis with Low-Dimensional Models:
Principles, Computation, and Applications

John Wright and Yi Ma,   Cambridge University Press


Basic Information

This book is to be used as an introductary graduate textbook for the areas of data science, signal processing, optimization, and machine learning. It has been used for the course EECS 290 (Berkeley) and ELEN 6886 (Columbia).
